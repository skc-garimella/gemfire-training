Pivotal GemFire® Programming Examples Documentation

Environment Settings A guide to setting up your environment to run the examples.
Examples Sample GemFire Java applications:
  • cacheOverflow: a program that demonstrates how the logical amount of data stored in a GemFire cache region can exceed the amount of heap space in a Java Virtual Machine.
  • cacheRunner: a program that loads a cache.xml file and interactively runs caching commands.
  • clientAPI: shows client configuration through the API for a producer client and a consumer client.
  • clientServerConfigs: provides configuration files for creating a client/server caching system. This example also demonstrates GemFire authentication and authorization.
  • haOverflow: demonstrates HA queue overflow for durable clients.
  • delta: an application that demonstrates delta propagation.
  • transaction: an application that demonstrates how to use transactions with colocated partitioned regions.
  • portableDataExchange: an application that demonstrates how to serialize objects using the Portable Data eXchange framework.
  • partitionedPersistence: an application that demonstrates how to use partitioned regions with disk persistence.
  • partitionedRegionQuerying: an application that demonstrates how to query partitioned regions using a function.
